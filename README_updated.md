## 🚀 Overview

The **Multi-Agent Education System** is an AI-powered platform that
generates structured study material for any topic using a multi-agent
architecture.

------------------------------------------------------------------------

## 🎯 Problem Statement

Students face: - Information overload\
- Difficulty finding relevant content\
- Time-consuming research

------------------------------------------------------------------------

## ✨ Features

-   Automatic Study Guide Generation\
-   Multi-Agent System (CrewAI)\
-   Fast Processing using Groq LLM\
-   Structured Content\
-   Streamlit UI\
-   Markdown Storage

------------------------------------------------------------------------

## 🛠️ Tech Stack

-   Frontend: Streamlit\
-   Backend: Python\
-   AI Framework: CrewAI\
-   LLM Integration: Groq\
-   Storage: Markdown Files

------------------------------------------------------------------------

## 📊 Results

-   Generates well-structured study guides\
-   Reduces manual research effort\
-   Provides fast and efficient output\
-   Demonstrates real-world use of multi-agent AI
